Ponderosa Pine Forest Simulation

Andrew Sánchez Meador

2026-02-10

Overview

This vignette demonstrates how to create custom configurations for forest types other than pinyon-juniper. We simulate a ponderosa pine dominated mixed-conifer forest with:

  • Custom allometric parameters for larger trees
  • Multiple species (PIPO, PSME, ABCO)
  • Regular spacing (managed forest)
  • No nurse tree effect

Why Custom Configurations?

The pre-built pj_huffman_2009() configuration is designed for pinyon-juniper woodlands. Other forest types differ substantially:

Characteristic Pinyon-Juniper Ponderosa Pine
Density ~900 trees/ha ~450 trees/ha
Mean DBH 20 cm 35 cm
Mean Height 6.5 m 18 m
Species 2 (PIED, JUSO) 3+ (PIPO, PSME, ABCO)
Spacing Random (R ~1.0) Regular (R ~1.4)
Nurse Effect Yes No

Step 1: Load Package

library(EmpiricalPatternR)

Step 2: Examine Ponderosa Allometric Parameters

Species-specific allometric equations control crown, height, and foliage relationships. The package ships with a ponderosa pine parameter set:

ponderosa_params <- get_ponderosa_allometric_params()

# Crown diameter model: ln(CD) = a + b*ln(DBH) + c*ln(H)
ponderosa_params$crown_diameter$PIPO
#> $a
#> [1] -0.5
#> 
#> $b
#> [1] 0.3
#> 
#> $c
#> [1] 0.4

# Height model: H = 1.3 + a*(1 - exp(-b * DBH))
ponderosa_params$height$PIPO
#> $a
#> [1] 35
#> 
#> $b
#> [1] 0.025

# CBH method
ponderosa_params$cbh_method
#> [1] "simple_ratio"

Compare tree dimensions for a 40 cm DBH ponderosa pine under different parameter sets:

test_dbh <- 40
test_sp  <- "PIPO"

# Default (P-J) parameters
pj_params <- get_default_allometric_params()
pj_h  <- calc_height(test_dbh, test_sp, pj_params)
pj_cr <- calc_crown_radius(test_dbh, pj_h, test_sp, pj_params)

# Ponderosa parameters
pp_h  <- calc_height(test_dbh, test_sp, ponderosa_params)
pp_cr <- calc_crown_radius(test_dbh, pp_h, test_sp, ponderosa_params)

cat(sprintf(
  "40 cm DBH Ponderosa Pine:\n  P-J equations:   Height = %.1f m, Crown radius = %.2f m\n  PP equations:    Height = %.1f m, Crown radius = %.2f m\n",
  pj_h, pj_cr, pp_h, pp_cr
))
#> 40 cm DBH Ponderosa Pine:
#>   P-J equations:   Height = 11.3 m, Crown radius = 2.47 m
#>   PP equations:    Height = 23.4 m, Crown radius = 3.24 m

Step 3: Build a Custom Configuration

Use create_config() to assemble all parameters:

config <- create_config(
  name = "Ponderosa Pine Mixed-Conifer",
  targets = list(
    density_ha    = 450,
    species_props = c(PIPO = 0.70, PSME = 0.20, ABCO = 0.10),
    species_names = c("PIPO", "PSME", "ABCO"),
    mean_dbh      = 35.0,
    sd_dbh        = 12.0,
    mean_height   = 18.0,
    sd_height     = 6.0,
    canopy_cover  = 0.45,
    cfl           = 0.85,
    clark_evans_r = 1.4
  ),
  weights = list(
    ce           = 30,
    dbh_mean     = 15,
    dbh_sd       = 10,
    height_mean  = 10,
    height_sd    = 5,
    species      = 60,
    canopy_cover = 60,
    cfl          = 50,
    density      = 80,
    nurse_effect = 0
  ),
  simulation = list(
    plot_size       = 100,
    max_iterations  = 5000,
    temp_initial    = 0.01,
    temp_final      = 0.0001,
    cooling_rate    = 0.9999,
    plot_interval   = 500,
    verbose         = TRUE,
    enable_plotting = TRUE,
    nurse_distance  = 0,
    use_nurse_effect = FALSE,
    mortality_prop  = 0.10
  ),
  allometric_params = ponderosa_params
)

print_config(config)

Step 4: Run Simulation

set.seed(456)

result <- simulate_stand(
  targets         = config$targets,
  weights         = config$weights,
  plot_size       = config$simulation$plot_size,
  max_iterations  = config$simulation$max_iterations,
  initial_temp    = config$simulation$temp_initial,
  cooling_rate    = config$simulation$cooling_rate,
  verbose         = TRUE,
  plot_interval   = 500,
  save_plots      = FALSE,
  nurse_distance  = config$simulation$nurse_distance,
  use_nurse_effect = config$simulation$use_nurse_effect,
  mortality_prop  = config$simulation$mortality_prop
)

Step 5: Analyze Results

# Console summary
print_simulation_summary(result)

# Comprehensive analysis with CSV and PDF output
analyze_simulation_results(
  result              = result,
  targets             = config$targets,
  prefix              = "ponderosa_forest",
  save_plots          = TRUE,
  nurse_distance_target = 0,
  target_mortality    = config$simulation$mortality_prop * 100
)

Output Files

When save_plots = TRUE, analyze_simulation_results() writes:

File Contents
*_all_trees.csv All trees (live + dead) with full attributes
*_live_trees.csv Live trees only
*_history.csv Optimization convergence history
*_summary.csv Summary statistics (simulated vs. target)
*_plots.pdf Spatial pattern and diameter distribution plots
