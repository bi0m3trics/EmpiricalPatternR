# Plot progress during simulation - showing objective progress

Plot progress during simulation - showing objective progress

## Usage

``` r
plot_progress(
  trees,
  metrics,
  targets,
  history,
  iter,
  energy,
  temperature,
  plot_size,
  save_plot = FALSE
)
```

## Arguments

- trees:

  Current trees data

- metrics:

  Current metrics

- targets:

  Target parameters

- history:

  History data table

- iter:

  Current iteration

- energy:

  Current energy

- temperature:

  Current temperature

- plot_size:

  Size of the plot area

- save_plot:

  Whether to save plot to file
