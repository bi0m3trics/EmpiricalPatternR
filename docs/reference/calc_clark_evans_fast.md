# Fast Spatial Pattern Metrics

Optimized calculation of Clark-Evans R using efficient distance
computation.

## Usage

``` r
calc_clark_evans_fast(x, y, plot_size)
```

## Arguments

  - x:
    
    Vector of x coordinates

  - y:
    
    Vector of y coordinates

  - plot\_size:
    
    Plot size (m)

## Value

Clark-Evans R value
