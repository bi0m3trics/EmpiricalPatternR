# Change species of a random tree

Change species of a random tree

## Usage

``` r
perturb_species(trees, species_names, species_probs)
```

## Arguments

  - trees:
    
    data.table. Tree data

  - species\_names:
    
    Character vector. Available species codes

  - species\_probs:
    
    Numeric vector. Target species proportions

## Value

data.table. Modified tree data
