# Get default allometric parameters for ponderosa pine forest

Example parameter set for ponderosa pine dominated forests. Demonstrates
how to create custom allometric relationships.

## Usage

``` r
get_ponderosa_allometric_params()
```

## Value

List of allometric parameters for ponderosa pine

## Examples

``` r
# Use ponderosa pine parameters
params <- get_ponderosa_allometric_params()
```
